# 📝 GreenClaw 更新日誌

## 🎉 v1.0.0 - 正式版（含圖片、搜索、社群小編）

### ✨ 功能

- **多 AI Provider**：OpenAI、Groq、OpenRouter，動態模型選擇
- **圖片生成**：DALL-E 整合（Stable Diffusion 預留）
- **網絡搜索**：Bing API 整合（Google 預留）
- **社群小編模式**：一鍵生成文案、配圖、熱門話題
- **社交媒體**：TikTok、Facebook、Instagram、X (Twitter)、YouTube 預留結構
- **安全**：預設沙盒、視覺化權限、config 本地儲存

### 🛠️ 技術

- Express.js、OpenAI SDK、TailwindCSS
- 0 Code Web UI，無需 CLI / Docker

---

*GreenClaw 由 ClawBox 更名並擴充功能而來。*
